Global Video Games (Website)

Global Video Games is an educational website exploring the evolution of video games across different regions. The site highlights three representative games from each region, showcasing the first game, a current title, and a popular game from that area. I developed this project solo for a class assignment, creating the entire site using HTML, CSS, and AWS S3 for hosting. In addition to the game showcases, the website includes a section outlining the development process and design decisions I made while building it.

Tools: AWS S3, HTML, CSS
